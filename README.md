# 🎯 AItodo v0.2 - 智能待办事项管理助手

## ✨ 应用简介

**AItodo** 是一款基于 AI 大模型的智能待办事项管理工具，它能够通过自然语言和截图识别自动提取任务信息，让任务管理变得更加高效和智能。

### 🌟 核心功能

#### 1. **AI 智能识别**
- **文本解析**：支持自然语言输入，AI 自动提取任务标题、截止时间和描述
  - 例如："明天下午3点开会"会自动解析为具体任务
  - 智能识别相对时间（明天、下周二、三天后等）
- **截图识别**：通过 Ctrl+Alt+S 全局快捷键快速截图识别任务
  - 实时预览选择区域（选中区域正常亮度，未选中区域半透明变暗）
  - 十字光标精准选择
  - 支持 ESC 取消操作

#### 2. **智能任务管理**
- **多维度任务筛选**：
  - 今日待办
  - 明日待办
  - 未来7天
  - 所有待办
  - 已完成/过期任务
- **任务优先级**：支持标记重要任务，重要任务自动置顶显示
- **任务状态管理**：一键完成、永久删除、详情编辑

#### 3. **智能提醒系统**
- 📅 **提前1天提醒**：任务到期前24小时自动推送桌面通知
- ⚠️ **紧急提醒**：任务即将到期（1小时内）发送紧急通知
- ⭐ **重要任务强化提醒**：重要任务通知带特殊标识

#### 4. **系统级体验优化**
- **系统托盘**：ESC 键快速最小化到托盘，不占用任务栏
- **全局快捷键**：Ctrl+Alt+S 随时随地截图识别任务
- **开机自启动**：支持开机自动启动，后台默默守护
- **现代化 UI**：基于 CustomTkinter 的美观界面，支持深色模式
